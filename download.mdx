CloudBook is available as a native app for Android and Windows, both built with Flutter. You can also use the full-featured web app from any browser without installing anything.

All versions share the same account — your encrypted notes sync automatically across every device you log in to.

## Choose your platform

  <Card title="Android" icon="android" href="https://list-app.vercel.app/download">

  </Card>
  <Card title="Windows" icon="windows" href="https://list-app.vercel.app/download">
    Run CloudBook as a native desktop app on Windows. Write and edit notes in a dedicated window without opening a browser.
  </Card>
  <Card title="Web app" icon="globe" href="https://list-app.vercel.app">
    No installation needed. Open CloudBook in any modern browser and access all features immediately. Works on any operating system.
  </Card>

Visit [list-app.vercel.app/download](https://list-app.vercel.app/download) to download the Android APK or Windows installer.

<Note>
  Native app downloads are available directly from the CloudBook website. The Android and Windows apps are built with Flutter and are not currently distributed through app stores.
</Note>

## Platform comparison

| Feature | Web | Android | Windows |
|---|---|---|---|
| No installation required | Yes | No | No |
| Offline note-taking | No | Yes | Yes |
| Automatic sync | Yes | Yes | Yes |
| AES encryption | Yes | Yes | Yes |
| Rich text editor | Yes | Yes | Yes |

## Get started after downloading

Once you've installed the app, open it and log in with your CloudBook account. If you don't have an account yet, see the [quickstart guide](/quickstart) to create one.

<Tip>
  If you've already used CloudBook on the web, log in with the same credentials on your native app. Your notes will appear immediately after you sign in.
</Tip>
